# The Launch

RSA training and reference app for Ashley Furniture stores.

Built for the 5th & Home franchise — Corpus Christi, Victoria, Bay City, 5th & Home.

## What's in the box

- 6-day onboarding program (Foundation → Discovery → Presentation → Closing → System & Money → Specialization)
- 19 cheat sheets covering EASY framework, bedding, SafeLock, finance partners, the Up Board, HOMES POS, RSA math, and more
- Pay calculator with bonus qualification tracking
- Multi-profile system with PIN protection for shared devices
- Pre-training assessment that emails results to the trainer
- Quiz scores and completion reports sent via email to the trainer

## Deployment

This is a Next.js app deployed on Vercel.
